# 变量类型
JavaScript变量类型分为值类型和引用类型

## 值类型
数字，字符串，布尔，symbol, undefined
## 引用类型
对象(数组，函数，null)
## 相关问题
1. typeof可以判断的类型
可以识别所有值类型
可以识别函数
无法细分其它引用类型（object）
2. === 和 == 的区别
3. 值类型和引用类型的区别
4. 实现深拷贝
```javascript
function deepClone (obj) {
    if (typeof obj !== 'object' || obj == null) {
        return obj
    }
    let res
    if (Array.isArray(obj)) {
        res = []
    } else {
        res = {}
    }
    for (key in obj) {
        if (obj.hasOwnProperty(key)) {
            res[key] = deepClone(obj[key])
        }
    }
    return res
}
```
